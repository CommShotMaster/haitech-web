我们的课程

引人入胜的机器人课程

专为小学生量身定制的沉浸式教育体验。

🏕️

一日学校营

充满机器人乐趣的充实一天！学生将接触到激动人心的编程和工程世界。他们将学习编程基础、组装自己的机器人，并在一天结束时进行一场令人兴奋的机器人比赛。非常适合学校的课外活动或假日安排！

🏫

拓展课程

我们的持续拓展课程带您深入了解 STEAM。我们与部分小学合作开设这些进阶课程，带领学生从可视化图形编程一路进阶到 Python 入门。学生将在学期中建立起逻辑和技术的坚实基础。

教学大纲

您的孩子将学到什么

🧩

可视化图形编程

使用基于 Scratch 的可视化编程界面，初学者可以轻松拼接代码块来指挥他们的机器人，掌握编程逻辑，而无需担心遇到复杂的语法错误。

🐍

Python 入门

随着学生的进步，他们将过渡到编写实际的文本 Python 代码，让他们在世界上最受欢迎的编程语言中获得领先优势。

⚙️

应用物理与数学

学生们将实时运用速度、距离、角度和几何等概念来引导机器人穿越迷宫、计算飞行轨迹并完成各种物理挑战。

常见问题

家长最关心的几个问题

几岁的孩子适合参加？

+
我们的课程专为 7 到 12 岁（小学阶段）的学生设计，内容既有基础的图形化编程，也有进阶的 Python 编程。

需要自己带电脑吗？

+
不需要！我们为每一位学生提供所需的全部设备，包括笔记本电脑、机器人和各类教具。

没有任何编程基础可以参加吗？

+
当然可以！我们的课程从零基础开始，通过直观的 Scratch 模块化编程，让孩子们轻松上手，逐步建立逻辑思维。

夏令营包午餐吗？

+
一日营包含健康的午餐和下午茶点心，您在报名时可以告知我们孩子是否有任何食物过敏史。
