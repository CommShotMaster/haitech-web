To push a website 
1. git init
2. git remote add origin https://github.com/CommShotMaster/haitech-web.git
3. git add .
4. git config --global user.email "qiesheng030207@gmail.com"
5. git config --global user.name "CommShotMaster"
6. git commit -m "first version"
7. git branch -M main
8. git push -u origin main --force
